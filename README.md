# Ejecutar el proyecto en desarrollo

1. Clonar el proyecto
   2.Ejecutar

```
npm install
```

3. Tener nest CLI instalado

```
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```
docker-compose up -d

```

5. Recargar la base de datos

```
localhost:3000/api/v2/seed
```

# Stack usado

- MongoBb
- NestJS
